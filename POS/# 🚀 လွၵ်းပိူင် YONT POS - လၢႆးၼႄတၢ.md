## 💻 လွင်ႈလူဝ်ႇ System

### ဢၼ်လူဝ်ႇလႆႈႁဵတ်း တီႈဢေႇသုတ်း -
- **လွၵ်းလၢႆးႁဵတ်းၵၢၼ်** : ဝိၼ်းတူဝ်း 10 ၶိုၼ်ႈၼိူဝ် (64-bit)
- **RAM**: တီႈဢေႇသုတ်း 4 GB
- **တီႈသိမ်းၶေႃႈမုၼ်း**: တီႈၼင်ႈလွတ်ႈလႅဝ်း 2 GB
- **ၼႄ**: လွင်ႈၼႄ 1024x768 တီႈဢေႇသုတ်း (ပၼ်တၢင်းႁၼ်ထိုင်ဝႃႈ 1920x1080 ဢမ်ႇၼၼ် သုင်လိူဝ်ၼၼ်ႉ တႃႇလွင်ႈတူဝ်ထူပ်းဢၼ်လီသုတ်း)
- **ဢိၼ်ႇတႃႇၼႅတ်ႉ**: ဢမ်ႇလူဝ်ႇၸႂ်ႉ (တွၼ်ႈတႃႇႁပ်ႉငိုၼ်း ဢီးမေးလ် လႄႈ ႁဵတ်း Backup)

### ၶေႃႈပၼ်တၢင်းႁၼ်ထိုင် ဢၼ်လူဝ်ႇႁဵတ်း -
- **လွၵ်းလၢႆးႁဵတ်းၵၢၼ်**: ဝိၼ်ႇတူဝ်ႇ 11 (64-bit)
- **RAM** : 8 GB ၶိုၼ်ႈၼိူဝ်
- **တီႈသိမ်းၶေႃႈမုၼ်း**: SSD (မီး 5 GB)
- **ၼႄ**: ၽွၼ်ႉ HD (1920x1080) ဢမ်ႇၼၼ် သုင်လိူဝ်
- **ၶိူင်ႈဢိတ်ႇလိၵ်ႈ**: ၶိူင်ႈဢိတ်ႇလိၵ်ႈ ဢၼ်ႁပ်ႉဝႂ်မႆႈ (ESC/POS ၵိုင်ႇငၢမ်ႇၵၼ်) ဢမ်ႇၼၼ် ၶိူင်ႈဢိတ်ႇလိၵ်ႈ ဢၼ်ပဵၼ်ပိူင်

## 📦 လွၵ်းလၢႆးတၢင်ႇ

### ၶၵ်ႉတွၼ်ႈ 1: လူတ်ႇဢဝ် Installer ၊

1. ဢဝ်ၾၢႆႇ **YONT_POS_Setup.exe** တီႈၽူႈၶၢႆၸဝ်ႈၵဝ်ႇ
2. သိမ်းဝႆႉတီႈ Desktop ဢမ်ႇၼၼ် Downloads ၸဝ်ႈၵဝ်ႇ

### ၶၵ်ႉတွၼ်ႈ 2 လွင်ႈႁၢင်ႈႁႅၼ်း ဢွၼ်တၢင်း တႃႇတေတင်ႈ

**လွင်ႈယႂ်ႇ**: ႁဵတ်းၸွမ်းၼင်ႇ ၶၵ်ႉတွၼ်ႈၸိူဝ်းၼႆႉသေ ႁႂ်ႈမၼ်ႈၸႂ်ဝႃႈ တေလႆႈတင်ႈ တီႈ **C:/YONT_POS** ႁႂ်ႈထုၵ်ႇမႅၼ်ႈ

1. **ႁဵတ်းပၼ် ၽူဝ်ႇတႃႇ တႃႇတၢင်ႇ**: 
- ၼဵၵ်း `ဝိၼ်ႇတူဝ်ႇ ၶီး + R` 
- ပေႃႉ: `C:\` သေ ၼဵၵ်း Enter 
- ၼဵၵ်းၶႂႃတီႈပဝ်ႇ → **မႂ်ႇ** → **ၽူဝ်ႇတႃႇ** 
- ပၼ်ၸိုဝ်ႈၽူဝ်ႇတႃႇ : `YONT_POS` 
- ဢၼ်ၼႆႉ ႁဵတ်းဢွၵ်ႇပၼ်: `C:\YONT_POS`

2. **တင်ႈပၼ်ၶႂၢင်ႉ Folder** (သင်ဝႃႈ ယွၼ်းမႃး) : 
- ၼဵၵ်းၶႂႃတီႈ `C:\YONT_POS` ၽူဝ်ႇတႃႇ 
- လိူၵ်ႈဢဝ် **ၶူဝ်းၶွင်** → **လွင်ႈႁူမ်ႇလူမ်ႈ** → **မႄးထတ်း** 
- လိူၵ်ႈၸိုဝ်ႈၽူႈၸႂ်ႉတိုဝ်းၸဝ်ႈၵဝ်ႇ 
- ၼဵၵ်းတီႈ **ၵုမ်းထိင်းတဵမ်ထူၼ်ႈ** → **ၸႂ်ႉတိုဝ်း** → **လီ**

### ၶၵ်ႉတွၼ်ႈ 3 : ႁဵတ်း Installer ၊

1. **ဢိုတ်းပႅတ်ႈ ဢႅပ်ႉဢၼ်ႁဵတ်းၵၢၼ်ယူႇတင်းမူတ်း** (ပိူင်လူင်မၼ်း ၶိူင်ႈထတ်းတူၺ်း ၸူဝ်ႈၶၢဝ်း ဢၼ်ႁႄႉၵင်ႈ မႅင်းဝၢႆးရၢတ်ႉသ်)
2. **ၼဵၵ်းသွင်ၵမ်း** `YONT_POS_တမ်းဝၢင်း.exe`
3. **Windows တေၸၢင်ႈၼႄပၼ် ၶေႃႈပၼ်ၾၢင်ႉ လွင်ႈႁူမ်ႇလူမ်ႈ**: 
- ၼဵၵ်း **"ၶေႃႈမုၼ်းထႅင်ႈ"** သင်ဝႃႈ ထုၵ်ႇထၢမ် 
- ၼဵၵ်း **"လႅၼ်ႈလႆႈၸိူင်ႉႁိုဝ်ၵေႃႈယဝ်ႉ"**

### ၶၵ်ႉတွၼ်ႈ 4 : ႁဵတ်းၸွမ်း ဝီႇၸႃႇတႃႇ တႃႇတင်ႈၶိူင်ႈ

**လွင်ႈတင်ႈၶိုၼ်ႈ ၶၵ်ႉတွၼ်ႈၼိုင်ႈယဝ်ႉၶၵ်ႉတွၼ်ႈၼိုင်ႈ**:

| ၶၵ်ႉတွၼ်ႈ | လွင်ႈႁဵတ်းသၢင်ႈ | ၶေႃႈမၢႆတွင်း |
|------|-------|-------|
| **ႁူမ်ၸူမ်းႁပ်ႉတွၼ်ႈ** | ၼဵၵ်း **ထႅင်ႈ** | ထတ်းတူၺ်းၶေႃႈမၵ်းမၼ်ႈ သင်ၼႄဝႆႉ |
| **သဵၼ်ႈတၢင်း တႃႇတင်ႈ** | ပေႃႉ ဢမ်ႇၼၼ် ၶူၼ်ႉႁႃ တီႈ: `C:\YONT_POS` | ⚠️ **လွင်ႈယႂ်ႇ**: ယႃႇၸႂ်ႉ `ၾၢႆႇပရူဝ်ႇၵရႅမ်ႇ` |
| **လိူၵ်ႈဢဝ် ၶူဝ်းၶွင်** | လိူၵ်ႈဢဝ်ၶိူင်ႈၶူဝ်းတင်းမူတ်း | ဝႆႉပိူင်တၢႆ ဢၼ်လိူၵ်ႈဝႆႉ |
| **တၢင်းလတ်း** | လိူၵ်ႈဢဝ် ဢၼ်ထုၵ်ႇၸႂ် | သင်ဝႃႈ ဢမ်ႇၶႂ်ႈလႆႈ လွၵ်းလၢႆးလတ်း တႃႇ Desktop/Start Menu ၼႆၸိုင် ၸၢင်ႈယႃႉပႅတ်ႈလႆႈ |
| **ႁၢင်ႈႁႅၼ်းတႃႇသႂ်ႇ** | ၼဵၵ်း **တၢင်ႇ** | တေတႄႇတင်ႈ |
| **ႁဵတ်းႁႂ်ႈယဝ်ႉတူဝ်ႈ** | ၼဵၵ်း **ယဝ်ႉတူဝ်ႈ** | ဢႅပ်ႉပလီႇၵေးသျိၼ်း တေၸၢင်ႈပိုတ်ႇႁင်းမၼ်း |

### ၶၵ်ႉတွၼ်ႈ 5: ထတ်းတူၺ်း လွင်ႈတင်ႈ

ဝၢႆးသေ တင်ႈယဝ်ႉတူဝ်ႈၵႂႃႇယဝ်ႉ-

1. **ထတ်းတူၺ်း ၽူဝ်ႇတႃႇ တႃႇတၢင်ႇ**: 
- ၼဵၵ်း `ဝိၼ်းတူဝ်ႇၶီး + E` (ၽူႈလဵပ်ႈႁဵၼ်းၾၢႆႇ) 
- ၼႄတၢင်းၵႂႃႇတီႈ: `C:\YONT_POS` 
- ထတ်းတူၺ်းဝႃႈ ၸဝ်ႈၵဝ်ႇႁၼ်လႆႈ : `main.exe`, `settings.json`, ၽူဝ်ႇတႃႇ ၾွၼ်ႉ၊

2. **ႁဵတ်း Backup ၶေႃႈမုၼ်း** (လွင်ႈယႂ်ႇ): 
- ႁဵတ်းၽူဝ်ႇတႃႇ `C:\YONT_POS\ၶႅပ်းႁၢင်ႈ` 
- ၽူဝ်ႇတႃႇၼႆႉ မၼ်းတေသိမ်းဝႆႉ ၶေႃႈမုၼ်း ဢၼ်ၸဝ်ႈၵဝ်ႇ ႁဵတ်း Backup ဝႆႉ ႁင်းမၼ်းၼၼ်ႉ

3. **ထတ်းတူၺ်း လွၵ်းလၢႆးလတ်း** (သင်ႁဵတ်းဝႆႉ): 
- ၶွမ်းၶွမ်း ထုၵ်ႇလီမီး **YONT POS** လွၵ်းလၢႆးလတ်း 
- တႄႇမႅၼ်ႇၼူဝ်ႇ → **YONT POS** ၽူဝ်ႇတႃႇ ဢၼ်ပႃး လွၵ်းလၢႆးလတ်း

### ၶၵ်ႉတွၼ်ႈ 6 : ပိုတ်ႇၽုၺ်ႇပွၵ်ႈဢွၼ်တၢင်းသုတ်း

1. **ၼဵၵ်းသွင်ၵမ်း** တီႈ **YONT POS** ဢၼ်မီးတီႈၼိူဝ်ၶွမ်း ဢမ်ႇၼၼ်
2. **ပိုတ်ႇ မႅၼ်ႇၼူဝ်ႇ တႄႇ** → ႁႃ **YONT POS** → ၼဵၵ်း **YONT POS**
3. ပႂ်ႉႁႂ်ႈဢႅပ်ႉပလီႇၵေးသျိၼ်း load မႃး (ပိုတ်ႇဢွၼ်တၢင်းသုတ်း တေၸၢင်ႈႁိုင် 5-10 ၸူဝ်ႈမူင်း)။

### လႅၵ်ႈလၢႆႈတီႈတင်ႈ

**သင်ဝႃႈ ၸဝ်ႈၵဝ်ႇ ဢမ်ႇတင်ႈၸႂ်သေ တၢင်ႇသႂ်ႇၼႂ်း Program Files** ၼႆၸိုင်၊

1. ဢဝ်ဢွၵ်ႇ : လွင်ႈတမ်းဝၢင်း → ဢႅပ်ႉ → ဢႅပ်ႉ လႄႈ လွင်ႈႁဵတ်းၵၢၼ် → ႁႃ **YONT POS** → **ထွၼ်ဢွၵ်ႇ**
2. မွတ်ႇပႅတ်ႈ : `C:\ၾၢႆႇပရူဝ်ႇၵရႅမ်ႇ\YONT_POS` ၽူဝ်ႇတႃႇ (သင်ဝႃႈမီးဝႆႉ)
3. ႁဵတ်းၸွမ်းၼင်ႇ လၢႆးတၢင်ႇ ဢၼ်လၢတ်ႈမႃးပႃႈၼိူဝ်ၼၼ်ႉသေတႃႉ လႅၵ်ႈလၢႆႈသဵၼ်ႈတၢင်းမၼ်း ႁႂ်ႈပဵၼ် `C:\YONT_POS` ။

**ယႃႇပေတၢင်ႇတီႈ**:
- ❌ `ၾၢႆႇပရူဝ်ႇၵရႅမ်ႇ` ဢမ်ႇၼၼ် `ၾၢႆႇပရူဝ်ႇၵရႅမ်ႇ (x86)` (ပၼ်ႁႃၶႂၢင်ႉ)
- ❌ ၶွမ်း (ၽေးၶေႃႈမုၼ်းလူႉသုမ်း)
- ❌ Folder ၸူဝ်ႈၶၢဝ်း (လွင်ႈႁဵတ်းႁႂ်ႈမူတ်းသႂ်ႁင်းၵူၺ်းမၼ်းတေမွတ်ႇပႅတ်ႈၶေႃႈမုၼ်း)

**ႁၢင်ႊၽၢင်**:
- ✅ ပိုတ်ႇၶၢဝ်းယၢမ်းမႂ်ႇ
- ✅ တူၺ်းၶၢဝ်းယၢမ်းဢၼ်တူင်ႉၼိုင်ယူႇ

- ✅ တူၺ်းၶၢႆ ၽွင်းၶၢဝ်း shift

- ✅ ဢိတ်ႇဢွၵ်ႇ ၶေႃႈၼႄ ၶၢဝ်းယၢမ်း

**ၽႂ်ၶဝ်ႈတူၺ်းလႆႈ**: ၽူႈၸတ်းၵၢၼ် ၽူႈၵုမ်းငိုၼ်း ၽူႈၸတ်းၵၢၼ်
**ၶေႃႈတၢင်ႇလဝ်ႈ**: ၵူၼ်းႁပ်ႉငိုၼ်း တေလႆႈပိုတ်ႇၶၢဝ်းယၢမ်း ၵွၼ်ႇပႆႇၶၢႆ

**လၢႆးၸႂ်ႉတိုဝ်း**:
1. ၼဵၵ်း **Shift** ၶႅပ်း
2. **ပိုတ်ႇ Shift**: သႂ်ႇငိုၼ်းတႄႇ → မၵ်းမၼ်ႈ
3. သိုပ်ႇႁဵတ်းၵၢၼ်ၶၢႆ (တေလႆႈမိူဝ်ႈ shift ႁဵတ်းၵၢၼ်ယူႇၼၼ်ႉၵူၺ်း)
4. **ဢိုတ်း Shift**: မိူဝ်ႈယဝ်ႉတူဝ်ႈ တႃႇဝၼ်းၼၼ်ႉ 
- သႂ်ႇငိုၼ်းသုတ်း 
- လွၵ်းပိူင်ၼပ်ႉသွၼ်ႇလွင်ႈဢမ်ႇမိူၼ်ၵၼ်

- ✅ ၸွမ်းတူၺ်း ၸၼ်ႉၶူဝ်းၶွင် ၼႂ်းၶၢဝ်းယၢမ်းတႄႉတႄႉ
- ✅ ၶေႃႈၾၢင်ႉသတၵ်ႉဢေႇ
- ✅ ပိုၼ်းသတၵ်ႉ/သဵၼ်ႈတၢင်းထတ်းသၢင်

- ✅ ႁဵတ်းဢွၵ်ႇ ၽိုၼ်တၢမ်းငဝ်းလၢႆး သတၵ်ႉ

**ၽႂ်ၶဝ်ႈလႆႈ**: ၽူႈၸတ်းၵၢၼ် ၽူႈၸတ်းၵၢၼ် ၶူဝ်းၶွင် ၽူႈၸတ်းၵၢၼ်
**လွင်ႈယႂ်ႇ**: သိမ်းဝႆႉ ၶူဝ်းၶွင် ဢၼ်မၢၼ်ႇမႅၼ်ႈ တႃႇၶၢဝ်ႇငၢဝ်း ဢၼ်မၢၼ်ႇမႅၼ်ႈ

**လၢႆးၸႂ်ႉတိုဝ်း**:
1. ၼဵၵ်းတီႈ **ၽူႈၸတ်းၵၢၼ်သတၵ်ႉ** ၶႅပ်း
2. သွၵ်ႈႁႃၶူဝ်းၶွင် လူၺ်ႈၸိုဝ်ႈ/ Barcode
3. **ထႅမ်သတၵ်ႉ**: 
- သႂ်ႇတၢင်းၼမ် 
- ထႅမ်ၶေႃႈမၢႆတွင်း (ဢမ်ႇလူဝ်ႇ) 
- ၼဵၵ်း Save ။
4. **ဢဝ်ဢွၵ်ႇ Stock**: 
- သႂ်ႇတၢင်းၼမ် တႃႇဢဝ်ဢွၵ်ႇ 
- လိူၵ်ႈလွင်ႈတၢင်း (လွင်ႈလူႉသုမ်း/လွင်ႈမႄးၶၢႆ/လွင်ႈလူႉသုမ်း) 
- ၼဵၵ်း Save ။
5. တူၺ်းပိုၼ်းၶူဝ်းၶွင်ၵူႈဢၼ်ဢၼ်

**ယိူင်းဢၢၼ်း**: ၸတ်းၵၢၼ် ၶေႃႈမုၼ်း ၽူႈၸႂ်ႉတိုဝ်း လႄႈ ပိုၼ်းသိုဝ်ႉၶၢႆ

**ႁၢင်ႊၽၢင်**:
- ✅ ထႅမ်ၽူႈသိုဝ်ႉၶၢႆမႂ်ႇ
- ✅ တူၺ်းၼႃႈၽူႈၸႂ်ႉတိုဝ်း
- ✅ ၸွမ်းတူၺ်း လွင်ႈသိုဝ်ႉၶၢႆ ၶွင်ၽူႈသိုဝ်ႉၶၢႆ

- ✅ ၶေႃႈမုၼ်း လွင်ႈမီးသတ်ႉၸႃႇ ၽူႈၸႂ်ႉတိုဝ်း

**ၽႂ်ၶဝ်ႈတူၺ်းလႆႈ**: ၽူႈၸတ်းၵၢၼ် ၽူႈၸတ်းၵၢၼ် ၽူႈၵုမ်းငိုၼ်း
**ၸႂ်ႉတိုဝ်း**: ဝႂ်ႁပ်ႉငိုၼ်း ဢီးမေးလ်၊ လွင်ႈၸွမ်းလင် လွင်ႈမီးသတ်ႉၸႃႇ၊ လွင်ႈၶၢႆၶွၼ်ႇ

**လၢႆးၸႂ်ႉတိုဝ်း**:

2. **ထႅမ်ၽူႈၸႂ်ႉတိုဝ်းမႂ်ႇ**:

- သႂ်ႇဢီးမေးလ် (တွၼ်ႈတႃႇသူင်ႇဝႂ်ႁပ်ႉႁွင်း)

- ထႅမ်သႂ်ႇ ၶေႃႈမၢႆတွင်း ဢၼ်ဢမ်ႇလူဝ်ႇ 
- ၼဵၵ်း Save ။
3. တူၺ်းပိုၼ်းသိုဝ်ႉၶူဝ်းၶွင် ၽူႈသိုဝ်ႉၶၢႆ
4. ႁဵတ်း Update ၶေႃႈမုၼ်း ၽူႈၸႂ်ႉတိုဝ်း

### ႁဵတ်းဢွၵ်ႇလႄႈ ၸတ်းၵၢၼ်ၽူႈၸႂ်ႉတိုဝ်း

**ပဵၼ် ADMIN ၵူၺ်း ၸတ်းၵၢၼ်လႆႈ ၽူႈၸႂ်ႉတိုဝ်း**

#### ထႅမ်ၽူႈၸႂ်ႉတိုဝ်းမႂ်ႇ:
1. ၼဵၵ်းတီႈ **Settings** ⚙️ .
2. ၵႂႃႇတီႈ **ၼႃႈတီႈၽူႈၸႂ်ႉတိုဝ်းလႄႈ သုၼ်ႇလႆႈ** ၼၼ်ႉ
3. ၼဵၵ်း **ထႅမ်ၽူႈၸႂ်ႉတိုဝ်းမႂ်ႇ**
4. တႅမ်ႈသႂ်ႇ - 
- **ၸိုဝ်ႈၽူႈၸႂ်ႉတိုဝ်း**: ၶေႃႈမၵ်းမၼ်ႈဢၼ်ဢမ်ႇမိူၼ်ၵၼ် 
- **မၢႆလပ်ႉ**: မၢႆလပ်ႉဢၼ်ႁႅင်းယႂ်ႇ (ပၼ်တၢင်းၶႆႈၸႂ် 8 တူဝ် တီႈဢေႇသုတ်း) 
- **ၸိုဝ်ႈၼႄ**: ၸိုဝ်ႈတဵမ်ထူၼ်ႈ တွၼ်ႈတႃႇလိၵ်ႈႁပ်ႉတွၼ်ႈ 
- **Role**: လိူၵ်ႈဢဝ် Admin/ၽူႈၵုမ်းငိုၼ်း/ၽူႈၸတ်းၵၢၼ်/ၽူႈၸတ်းၵၢၼ် ၶူဝ်းၶွင် 
- **ဢီးမေးလ်**: တွၼ်ႈတႃႇ လွင်ႈပၼ်ၶၢဝ်ႇ (ဢမ်ႇလူဝ်ႇ)
5. ၼဵၵ်း **ႁဵတ်းၽူႈၸႂ်ႉတိုဝ်း** ။

#### မူၼ်ႉမႄးၽူႈၸႂ်ႉတိုဝ်း:
1. ၼႂ်း **ၼႃႈတီႈၽူႈၸႂ်ႉတိုဝ်းလႄႈ သုၼ်ႇလႆႈ** ၼၼ်ႉ
2. ႁႃၽူႈၸႂ်ႉတိုဝ်းၼႂ်းသဵၼ်ႈမၢႆ
3. ၼဵၵ်းတီႈ **မႄးထတ်း**
4. လႅၵ်ႈလၢႆႈၶေႃႈမုၼ်း
5. ၼဵၵ်း **Save** ။

#### မွတ်ႇပႅတ်ႈၽူႈၸႂ်ႉတိုဝ်း:
1. လိူၵ်ႈၽူႈၸႂ်ႉတိုဝ်း
2. ၼဵၵ်း **မွတ်ႇပႅတ်ႈ** ။
3. မၵ်းမၼ်ႈမွတ်ႇပႅတ်ႈ (⚠️ ဢမ်ႇၸၢင်ႈၶိုၼ်းႁဵတ်းၶိုၼ်း)

#### လႅၵ်ႈလၢႆႈမၢႆလပ်ႉၽူႈၸႂ်ႉတိုဝ်း:
- ၽူႈၸႂ်ႉတိုဝ်း လႅၵ်ႈလၢႆႈလႆႈ ႁင်းၽႂ်ႁင်းမၼ်း - လွင်ႈတမ်းဝၢင်း → မၢႆလပ်ႉ → သႂ်ႇမၢႆလပ်ႉၵဝ်ႇ & မၢႆလပ်ႉမႂ်ႇ
- Admin ၸၢင်ႈမႄးလႆႈ : ၽႅၼ်ႇၼႃႈတီႈၽူႈၸႂ်ႉတိုဝ်း → လိူၵ်ႈၽူႈၸႂ်ႉတိုဝ်း → မႄးၶိုၼ်းမၢႆလပ်ႉ

### ဝႂ်ႁပ်ႉႁွင်း ဢီးမေးလ် ဢမ်ႇသူင်ႇ

**ပၼ်ႁႃ**:"ဢမ်ႇၸၢင်ႈသူင်ႇဢီးမေးလ်" ၽိတ်းပိူင်ႈ

**ၶေႃႈၵေႈၶႆႈ**:
1. လွင်ႈတမ်းဝၢင်း → လွင်ႈတမ်းဝၢင်း ဢီးမေးလ် → ထတ်းတူၺ်းဝႃႈ ဢၵွင်ႉ Gmail လႆႈၶပ်ႉဝႆႉယဝ်ႉႁိုဝ်
2. ထတ်းတူၺ်းဝႃႈ ၽူႈၸႂ်ႉတိုဝ်း မီးဢီးမေးလ် ၼႂ်းပရူဝ်ႇၾႆး
3. ထတ်းတူၺ်း လွင်ႈၵပ်းသိုပ်ႇ ဢိၼ်ႇတႃႇၼႅတ်ႉ
4. ၸၢမ်းတူၺ်းသူင်ႇ → ၽႅၼ်ႇတမ်းဝၢင်း ဢီးမေးလ်
5. ပေႃးၸႂ်ႉတိုဝ်း Gmail ၼႆ ႁႂ်ႈပိုတ်ႇပၼ် "Apps ဢၼ်ဢမ်ႇပေႃးႁူမ်ႇလူမ်ႈ"
6. ၸႂ်ႉ App Password (ဢမ်ႇၸႂ်ႉ Password Gmail ၵူႈပွၵ်ႈ) ။

### လွင်ႈႁဵတ်းသၢင်ႈထိူင်း

**ပၼ်ႁႃ**: ဢႅပ်ႉပလီႇၵေးသျိၼ်း တူၵ်းလိုၼ်း ဢမ်ႇၼၼ် ၵွႆ

**ၶေႃႈၵေႈၶႆႈ**:
1. ဢိုတ်းပႅတ်ႈ ပရူဝ်ႇၵရႅမ်ႇတၢင်ႇဢၼ် (လူတ်းယွမ်းၸႂ်ႉတိုဝ်း RAM)
2. လွင်ႈတမ်းဝၢင်း → လွင်ႈၶိုၼ်ႈယႂ်ႇ → လွင်ႈႁဵတ်းႁႂ်ႈ ၶေႃႈမုၼ်း လီလိူဝ်မႃး
3. မွတ်ႇပႅတ်ႈ ၶေႃႈမုၼ်း : လွင်ႈတမ်းဝၢင်း → လွင်ႈၶိုၼ်ႈယႂ်ႇ → မွတ်ႇပႅတ်ႈ ၶေႃႈမုၼ်း ဢႅပ်ႉပလီႇၵေးသျိၼ်း
4. ၶိုၼ်းပိုတ်ႇဢႅပ်ႉ
5. ပေႃးႁဵတ်းၵၢၼ်ပရူဝ်ႇၵရႅမ်ႇၼမ်ၼႆ ႁႂ်ႈႁဵတ်း Upgrade RAM ။

**ပၼ်ႁႃ**: ၶူဝ်းၶွင်/ၶၢႆဢၼ်မွတ်ႇပႅတ်ႈ ၽိတ်းပိူင်ႈ

**ၶေႃႈၵေႈၶႆႈ**:

2. ၵပ်းသိုပ်ႇၽူႈၸတ်းၵၢၼ် တႃႇၶိုၼ်းႁဵတ်း Backup
3. **လွင်ႈႁႄႉၵင်ႈ**: ၵွၼ်ႇပႆႇမွတ်ႇပႅတ်ႈ ႁႂ်ႈထတ်းတူၺ်းတႃႇသေႇ
4. ပိုတ်ႇပၼ် : လွင်ႈတမ်းဝၢင်း → လွင်ႈၶိုၼ်ႈယႂ်ႇ → လွင်ႈႁဵတ်း Backup ႁင်းမၼ်း (ၵူႈဝၼ်း)

## 📞 ပၼ်ႁႅင်းလႄႈ လွင်ႈမႂ်ႇသုင်

### ၸွႆႈထႅမ်ပၢႆးပၺ်ႇၺႃႇ

**တွၼ်ႈတႃႇပၼ်ႁႃ**:
1. ထတ်းတူၺ်း ၾၢႆႇၽိတ်းပိူင်ႈ: `C:\YONT_POS\ၽိတ်းပိူင်ႈ.တူၵ်း`
2. ၵပ်းသိုပ်ႇၽူႈၸတ်းၵၢၼ် System တီႈ- 
- လိၵ်ႈလိၵ်ႈၽိတ်းပိူင်ႈ 
- ၶၵ်ႉတွၼ်ႈဢၼ်ႁဵတ်းႁႂ်ႈၽိတ်းပိူင်ႈ 
- ၶႅပ်းႁၢင်ႈ လွင်ႈၽိတ်းပိူင်ႈ 
- ၶေႃႈၼမ်း ၾၢႆႇၽိတ်းပိူင်ႈ.log

### ႁဵတ်း Backup ၶေႃႈမုၼ်းၸဝ်ႈၵဝ်ႇ

**လွင်ႈယႂ်ႇ: ႁဵတ်း Backup Data တႃႇသေႇ**

#### ႁဵတ်း Backup လူၺ်ႈမိုဝ်း:
1. လွင်ႈတင်ႈၶိုၼ်ႈ → လွင်ႈၶိုၼ်ႈယႂ်ႇ → **ႁဵတ်းBackup ယၢမ်းလဵဝ်**
2. သိမ်းဝႆႉတီႈ: `C:\YONT_POS\ၶႅပ်းႁၢင်ႈ\`
3. သိမ်းဝႆႉ Backup လၢႆလၢႆဢၼ် (ထုၵ်ႇလီႁဵတ်းၵူႈဝူင်ႈ)

#### ႁဵတ်း Backup လူၺ်ႈတူဝ်ၵဝ်ႇ:
1. လွင်ႈတမ်းဝၢင်း → လွင်ႈၶိုၼ်ႈယႂ်ႇ
2. ပိုတ်ႇပၼ် : "ဢူဝ်ႇတူဝ်ႇ Backup" ။
3. တင်ႈၶၢဝ်းယၢမ်း - ၵူႈဝၼ်း/ၵူႈဝူင်ႈ
4. ၶေႃႈမုၼ်း ဢၼ်သိမ်းဝႆႉၼၼ်ႉ တေသိမ်းဝႆႉ ႁင်းမၼ်း တီႈ: `C:\YONT_POS\ၶေႃႈမုၼ်း\`

#### ၶိုၼ်းဢဝ်မႃးတီႈ Backup:
1. လွင်ႈတင်ႈၶိုၼ်ႈ → လွင်ႈၶိုၼ်ႈယႂ်ႇ → **ၶိုၼ်းဢဝ်မႃးတီႈ Backup**
2. လိူၵ်ႈဢဝ် ၾၢႆႇ Backup ။
3. မၵ်းမၼ်ႈၶိုၼ်း (⚠️ တေတႅမ်ႈသႂ်ႇၶေႃႈမုၼ်းယၢမ်းလဵဝ်)
4. ဢႅပ်ႉတေၶိုၼ်းပိုတ်ႇၶိုၼ်း

### လွင်ႈထတ်းတူၺ်းၶေႃႈမုၼ်း System

**တႃႇတူၺ်းၶေႃႈမုၼ်း system လႄႈ ဝႅပ်ႇယၢင်ႇ**:

ၸၼ်ႉထၢၼ်ႈ (တႂ်ႈၽႅၼ်ႇတူၺ်း):
- ၼႄပၼ်ၽႃႇသႃႇဢၼ်ၸႂ်ႉယူႇယၢမ်းလဵဝ်
- ငဝ်းလၢႆးၵပ်းသိုပ်ႇ
- ၸၼ်ႉတူဝ်ႈတၼ်း

မႅၼ်ႇၼူဝ်ႇပႃႇ:
- ၶေႃႈမုၼ်း ဝႅပ်ႇယၢင်ႇ တီႈၼႂ်းတွၼ်ႈ About (ပေႃးမီး)

## 🎯 ၽိုၼ်ဢိင်ဝႆးဝႆး ၵၢၼ်ဢၼ်ၵႆႉႁဵတ်း

### ႁဵတ်းၵၢၼ်ၶၢႆ ႁႂ်ႈတဵမ်ထူၼ်ႈ
1. ၼဵၵ်း **ၶၢႆ**
2. သွၵ်ႈႁႃ customer (ဢမ်ႇၼၼ် ႁဵတ်းမႂ်ႇ)
3. သၵ်ႈၵႅၼ်ႇ ဢမ်ႇၼၼ် လိူၵ်ႈၶူဝ်းၶွင်
4. မူၼ်ႉမႄးတၢင်းၼမ်
5. ယွၼ်းလူတ်းယွမ်း (ပေႃးလူဝ်ႇ)
6. ပၼ်ငိုၼ်းတဵမ်ထူၼ်ႈ
7. ဝႂ်ႁပ်ႉႁွင်း ဢိတ်ႇဢွၵ်ႇ/ဢီးမေးလ်

### ထႅမ်ၶူဝ်းၶွင်မႂ်ႇ
1. ၼဵၵ်းတီႈ **ၶူဝ်းၶွင်**
2. ၼဵၵ်း **ထႅမ်မႂ်ႇ**
3. သႂ်ႇပၼ် ၸိုဝ်ႈ ၵႃႈၶၼ် မဵဝ်းသႅၼ်း ၶူဝ်းၶွင်
4. ထႅမ်ၶႅပ်းႁၢင်ႈ (ဢမ်ႇလူဝ်ႇ)
5. ၼဵၵ်း **Save** ။

### ပိုတ်ႇၶၢဝ်းယၢမ်း (ၽူႈၵုမ်းငိုၼ်း)
1. ၼဵၵ်းတီႈ **Shift** ၼၼ်ႉတႃႉ။
2. ၼဵၵ်း **ပိုတ်ႇ Shift** ။
3. သႂ်ႇငိုၼ်းတႄႇ
4. မၵ်းမၼ်ႈ

### ဢိုတ်းၶၢႆႉ
1. ၼဵၵ်းတီႈ **Shift** ၼၼ်ႉတႃႉ။
2. ၼဵၵ်း **ဢိုတ်း Shift** ။
3. သႂ်ႇငိုၼ်းဢၼ်သုတ်း
4. ထတ်းသၢင်ၶိုၼ်းလွင်ႈဢမ်ႇမိူၼ်ၵၼ်
5. မၵ်းမၼ်ႈ

### ႁဵတ်းဢွၵ်ႇၶေႃႈမုၼ်း
1. ၼဵၵ်း **ၶၢဝ်ႇငၢဝ်း**
2. လိူၵ်ႈဢဝ် မဵဝ်းၶေႃႈမုၼ်း
3. လိူၵ်ႈဝၼ်းထိ
4. ၼဵၵ်း **ႁဵတ်းဢွၵ်ႇ** ။
5. သူင်ႇဢွၵ်ႇၸွမ်းၼင်ႇလူဝ်ႇ

### လႅၵ်ႈလၢႆႈမၢႆလပ်ႉ
1. ၼဵၵ်းတီႈ **Settings** ⚙️ .
2. ၵႂႃႇတီႈ **မၢႆလပ်ႉ** ၼၼ်ႉ
3. သႂ်ႇမၢႆလပ်ႉၵဝ်ႇ
4. သႂ်ႇမၢႆလပ်ႉမႂ်ႇသွင်ပွၵ်ႈ
5. ၼဵၵ်း **Save** ။

### လႅၵ်ႈလၢႆႈၽႃႇသႃႇ
1. ၼဵၵ်းတီႈ **Settings** ⚙️ .
2. ၵႂႃႇတီႈ **ၽႃႇသႃႇ** ၼၼ်ႉ
3. လိူၵ်ႈ: ဢိင်းၵလဵတ်ႈ / ဆန် (Shan) / မြန်မာ (မျၢၼ်ႇမႃႇ)

5. ဢိၼ်ႇတႃႇၾႅတ်ႉ လႅၵ်ႈလၢႆႈၵမ်းလဵဝ်
